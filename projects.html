This appfeatures a map of the shootings that have occurred in Wilmington, DE from 2011 to the present. The frontend was designed using HTML, CSS, and JavaScript, while Python scripts were written that collect and update the data in the backend.

The data comes from Wilmington Shootings. Summary statistics and police reports are scraped from this site. Dates and addresses are then parsed out from the police reports, and the addresses are passed to Googleâ€™s Geocoding API to find the latitudes and longitudes used to plot the data. A Python script runs daily that updates the app with any new data that gets published on Wilmington Shootings.
